This simple program demonstrates how EES can be used to solve simultaneous non-linear equations.  The order that the equations appear is irrelevant.  Variables may appear anywhere in the equations. 

 

Be sure to use a * to signify multiplication and enter each equation on a separate line.

 

EES does not have any ‘built-in’ physics knowledge.  EES will happily solve “incorrect” equations as well as the “correct” equations so be sure that the equations are entered as you intended.
